# ProjExD_Groupe13

## 概要
前回の授業で作成した鳥無双を改良する。

## 仕様を固めようそう！そう「しよう」（やったこと）
1. ボス　（若林）
    1分に一つ出現。
    おっきい。
    たくさん球を出す。
    敵の出没回数が増えるなどの難易度調整した。

3. アイテム　（田端）
   敵を撃破すると10回に一度敵の場所に星が出没する。
   星を取得すると100点追加される。

4. 音の実装　（山田）
    たまの発射音
    被弾音の追加
    BGMを再生

5. こうかとん残機　（織田）
    被弾するとハートがひとつ減る。
    画面上部にハートを表示

6. jsonファイルに最高点を記載。ワールドレコードをクラウドにアップロード　（村田）
ゲームオーバー時に最高得点も同時に表示する。
最高点を更新したら誉める。
